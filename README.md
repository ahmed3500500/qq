# Crypto Signals (Android Kotlin)

- Android-only (Kotlin + Jetpack Compose)
- No login
- Connects to your server: http://37.49.228.169:8000
- Tabs: Opportunities / Latest / Settings
- Background notifications every 15 minutes using WorkManager
- Arabic + English (system language)

## Open in Android Studio
1) Open folder `CryptoSignalsApp`
2) Let Gradle sync
3) Run on device

## API Endpoints used
- GET /health
- GET /signals/latest?limit=50
- GET /signals/opportunities?limit=50
